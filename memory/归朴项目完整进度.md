### 模型配置
```yaml
model:
  default: minimax-cn/MiniMax-M2.5
  fallback: bailian/qwen3.5-plus
  routes:
    - pattern: ".*编程.*|.*代码.*"
      model: bailian/qwen3-coder-plus
    - pattern: ".*写作.*|.*文章.*"
      model: bailian/qwen3-max-2026-01-23
    - pattern: ".*设计.*|.*UI.*"
      model: bailian/kimi-k2.5
```

### 环境变量
```bash
export MINIMAX_API_KEY="sk-cp-..."
export FLASK_PORT=5000
export FLASK_HOST=0.0.0.0
export DEBUG=true
```

### 定时任务
```bash
# 每小时自动运行
0 * * * * cd /归朴/return && python3 scripts/schedule_reports.py >> logs/schedule.log 2>&1

# 心跳：每个小时整点
# 周报：每周日 22:00
# 月报：每月最后一天 22:00
```
